# Tg_bot with docker bild

## Description 
Project to study volume in docker


## Installation

``` shell
  
# clone repo:
git clone https://github.com/SashaMogilevskii/tg_bot_with_docker_bild.git


# create virtualenv:
virtualenv myenv

# activate virtualenv:
myenv/Scripts/activate


# create tg_token
# create .env file  and add your token in SECRET_TOKEN


# run tg_bot.py:
run main.py

```
